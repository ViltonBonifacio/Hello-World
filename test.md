## Code

### Code with `4 spaces` or `2 tabs`:
    function foo() {
      a = 1;
      b = 2;
      return a + b;
    }

### Code with ` ``` ` (fencing):
```
function foo() {
  a = 1;
  b = 2;
  return a + b;
}
```

### Code with ` ``` ` (fencing) + `language`:
```JavaScript
function foo() {
  a = 1;
  b = 2;
  return a + b;
}
```
